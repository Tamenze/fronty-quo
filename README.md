## :star2: About the Project
A Word is a platform for sharing and saving meaningful lines of prose. You can browse what resonates with others and curate what resonates with you. It's deployed at <a href="https://sayword.dev/">sayword.dev</a>.

### :space_invader: Tech Stack

**Client**
- [Typescript](https://www.typescriptlang.org/)
- [React.js](https://reactjs.org/)
- [TailwindCSS](https://tailwindcss.com/)
- [Shadcn UI Components](https://ui.shadcn.com/)
- [Tanstack Query](https://tanstack.com/query/latest)

**Server**
- [Ruby on Rails](https://rubyonrails.org/)

**Database**
- [PostgreSQL](https://www.postgresql.org/)
- [Supabase](https://supabase.com/) (managed Postgres)

**DevOps / Hosting**
- Frontend: [Netlify](https://www.netlify.com/)
- Backend: [Render](https://render.com/)
